## TMUX config

### Installation
```
git clone https://github.com/zulonas/tmux.git ~/.tmux
ln -sf ~/.tmux/tmux.conf ~/.tmux.conf
ln -sf ~/tmux/ ~/.tmux_conf_dir
# and reopen terminal
```
